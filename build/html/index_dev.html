Description¶

ZotPie is a open source plugin for Zotero (Firefox) that was created to give members of the Zotero community several features that have been long overdue.

Welcome to ZotPie’s developer documentation!¶

This is the developer documentation for the ZotPie firefox plugin for Zotero. If you are not a developer, you may be interested in the user documentation.

ZotPie is still under Development, developer documentation will be released as soon as a
stable version of Zotpie is realeased.

Thank You for your interest in developing w/ ZotPie.
