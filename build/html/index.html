Description¶

ZotPie is a open source plugin for Zotero (Firefox) that was created to give members of the Zotero community several features that have been long overdue.

Welcome to ZotPie’s documentation!¶

This is the user documentation for the ZotPie firefox plugin for Zotero. If you are a developer, you may be interested in the developer documentation.

Getting Started (short version)¶

  1. Download the plugin from ZotPie Download Page or GitHub .
  2. Install Zotero for FireFox
  3. Open FireFox and Navigate to the Addons Menu
  4. In settings (the gear icon w/ dropdown) select install addon from file and select the plugin
  5. Restart Firefox and launch Zotero for Firefox.

Installing, Testing, Usage (Longer Version)¶

Detailed Installation Instructions¶

  1. Download the plugin from #downloadpage (idk yet), link to git install instructions if git is installer
  2. Install Firefox & the Zotero FireFox Plugin
  3. Open firefox and navigate to the dropdown menu and then click on Add-ons.
  4. Click on the settings gearbox and then install addon from file
  5. Navigate to the .xpi file downloaded in the earlier steps and click OK
  6. restart Firefox, if not prompted. Zotero should now have the plugin installed.

Installing development versions¶

ZotPie remains in development as of November 2015. Development Versions can be found on the GitHub dev branch. You can also clone directly from a git enabled console by pasting: git clone http://github.com/repo

Building Documentation¶

If you wish to build ZotPie’s documentation for offline use, it can be built from the doc directory of a local git repo by running make followed by the desired output format(s) (html, epub, latexpdf, etc.)

Batch Editing Tags¶

This feature allows you to edit tags en masse, there are two ways to do this. The first is to open the batch editor (in Zotero) and then search for the tag you would like to change,delete,modify, or merge and click apply. The second is to manually add items to a batch editing queue Batch Editing: brute force and then open the batch editor and apply the changes to items in your queue.

Warning

The batch editor does not backup any data before making changes, the only way to undo changes to your library made by the batch editor are:
  1. backup your library before using the batch editor
  2. use the batch editor again, with the tag parameters switch
